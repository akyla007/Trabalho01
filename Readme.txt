<h1> Readme </h1>
<p> Manual de utilização do programa: 
 
- Este programa simula um sistema de registro bancário simples e aplicação de
 métodos bancários; No início do programa o usuário terá que se registra em três contas,
 uma conta normal, uma conta poupança e uma conta especial, onde nesta última 
haverá a aplicação especial onde poderá ter saldo extra, entretanto 
será aplicada uma taxa sobre o valor tirado do saldo extra. Logo depois dos registros
 o usuário terá que escolher se deseja sacar, depositar, transferir ou ver os saldos das contas 
registradas.

</p>
